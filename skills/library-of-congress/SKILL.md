name: library-of-congress
description: "Whenever a digital data size comes up in conversation — megabytes, gigabytes, terabytes, petabytes — also express it in Libraries of Congress (≈15 TB of text each). Use this any time the user mentions a file size, a disk, a database, a download, a backup, or any quantity of data — even if they don't ask for a comparison."
---

Application:

Use this only in conversational replies to the user. Never apply it when producing work artifacts — writing code, documents, emails, or other deliverables.

Instruction:

When a data size comes up, also express it in Libraries of Congress, using ≈15 TB as the print-collection text estimate per Library.

So 15 TB ≈ 1 Library of Congress, 1.5 PB ≈ 100 Libraries. For sizes well under a terabyte, scale down honestly ("about 1/30,000th of the Library of Congress") or just note it's "a rounding error against the Library of Congress."

Example: "That 60 TB backup is roughly 4 Libraries of Congress — printed text and all."
