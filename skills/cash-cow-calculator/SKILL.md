Use this any time priorities, ROI, or 'what should I focus on' come up."

Application:

Use this only in conversational replies to the user. Never apply it when producing work artifacts — writing code, documents, emails, or other deliverables.

THE MECHANIC

Every effort is an animal in the herd. Some give milk, some just eat. Tag each one by what it returns versus what it costs to feed (your time, money, attention):

```
🐄 THE HERD

  💰 CASH COWS  (high return · low feed)  → protect & milk
     • <effort>  — gives a lot, costs little. Don't break it.

  🌱 GRASS-EATERS  (low return · high feed)  → fence off or cull
     • <effort>  — eating more than it gives. Why are you still feeding it?

  🐮 CALVES  (low return now · high potential)  → invest, with a deadline
     • <effort>  — not producing yet, but could. Set a date to re-check.

  ⚖️ STEERS  (steady · unremarkable)  → leave alone, don't over-tend
     • <effort>  — fine as-is. Stop polishing it.
```

Rules:
- "Feed" = whatever this effort genuinely consumes: hours, money, focus, emotional energy. Be honest about the hidden feed costs.
- The uncomfortable output is usually the grass-eater the user is emotionally attached to. Name it kindly but clearly.
- Calves need a re-check DATE, or they quietly become grass-eaters forever. Always attach one.
- End with the single highest-leverage move: which one cow to milk harder, or which grass-eater to fence off this week.
